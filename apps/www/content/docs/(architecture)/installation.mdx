Step-by-step guide to setting up Bedstack.

## Prerequisites

Before you begin, ensure you have the following installed:

- **[Bun](https://bun.sh)** (v1.1 or higher)
- **[Docker](https://www.docker.com/)** (for running the database locally)

## Installation

### 1. Clone the repository

Clone the Bedstack repository to your local machine:

```bash terminal
git clone https://github.com/yamcodes/bedstack.git
cd bedstack
```

### 2. Install dependencies

Bedstack uses Bun's built-in package manager. Run the following command in the root directory:

```bash terminal
bun install
```

### 3. Set up the environment variables

Copy the example environment files for the `conduit` app:

```bash terminal
cp apps/conduit/.env.example apps/conduit/.env
```

### 4. Start the database

Bedstack uses PostgreSQL. You can start a local instance using Docker Compose:

```bash terminal
bun run --cwd apps/conduit db:start
```

### 5. Run migrations and seed

Apply the database schema and seed it with initial data:

```bash terminal
bun run --cwd apps/conduit db:push
bun run --cwd apps/conduit db:seed
```

### 6. Start the development server

Now you can start the `conduit` backend in development mode:

```bash terminal
bun run --cwd apps/conduit dev
```

The API will be available at `http://localhost:3000/api`.
