Once you have [Bedstack installed](/docs/installation), you can start exploring its features.

## 1. Explore with Swagger

Bedstack comes with built-in Swagger documentation thanks to ElysiaJS.

1.  Start the `conduit` app: `bun run --cwd apps/conduit dev`
2.  Open your browser to [http://localhost:3000/swagger](http://localhost:3000/swagger)

You'll see a fully interactive API documentation where you can test all the endpoints.

## 2. Make your first request

You can also use `curl` or any API client to interact with the API.

### Get Tags

```bash terminal
curl http://localhost:3000/api/tags
```

### Response

```json
{
  "tags": ["reactjs", "angularjs", "dragons"]
}
```

## 3. End-to-End Typesafety

Bedstack's core strength is typesafety. Because it uses Elysia and ArkType, your API is typesafe from the database to the client.

### Defining a Schema with ArkType

In `apps/conduit/src/users/users.schema.ts`:

```typescript
import { type } from "arktype"

export const UserSchema = type({
    email: "string.email",
    username: "string",
    bio: "string|null",
    image: "string|null"
})
```

### Using it in Elysia

```typescript
import { Elysia, t } from 'elysia'
import { UserSchema } from './users.schema'

new Elysia()
  .post('/user', ({ body }) => {
    // body is automatically validated and typed!
    return body
  }, {
    body: UserSchema
  })
```

## 4. Database access with Drizzle

Bedstack uses Drizzle ORM for performant and typesafe database access.

```typescript
import { db } from "../database"
import { users } from "../database/schema"
import { eq } from "drizzle-orm"

const result = await db.select().from(users).where(eq(users.id, 1))
```
